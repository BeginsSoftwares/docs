## Bem-vindo

A API da 3X Pay Gateway permite que você integre facilmente nossos serviços de pagamento em sua aplicação. Com ela você pode criar transações, consultar status, fazer estornos e muito mais.

## Autenticação

Para autenticar as requisições à API, você precisa incluir duas chaves nos headers das requisições:

- `api_key`
- `api_secret`

Essas credenciais são únicas para cada conta e podem ser encontradas no painel administrativo da 3X Pay.

<Card
  title="Acessando Minhas Credenciais"
  icon="key"

>

<Warning>
  Mantenha suas credenciais em segurança e nunca as compartilhe. Em caso de comprometimento, revogue-as imediatamente no painel.
</Warning>
